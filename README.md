# 📚 Library Management System

## 🚀 Tech Stack
- Java
- Spring Boot
- MySQL
- Postman
- JWT Authentication

---

## 🔥 Features
- User Registration & Login (JWT)
- Book Management
- Issue & Return Books
- User Dashboard
- Secure APIs (Spring Security)

---

## 📡 APIs

### 🔐 Auth APIs
- POST /auth/register
- POST /auth/login

### 📚 Book APIs
- POST /api/books
- GET /api/books

### 📖 Issue APIs
- POST /api/issue
- POST /api/return

### 👤 User APIs
- GET /api/user/{id}/books
